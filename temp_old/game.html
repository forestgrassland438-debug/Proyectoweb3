Shop Help

📱 On Mobile:

  • Single tap: Select item
  • Double tap: Open buy/sell panel
  • Swipe: Browse items
  • View History: Shows all transactions

💻 On PC:

  • Left click: Select item
  • Click Interact: Open transaction panel
  • Mouse wheel: Scroll list and history
  • Enter: Confirm transaction in panel
  • WASD: Work normally while typing

⚙️ Features:

  • Limits: Only apply to daily purchases
  • Fee: Only applies to sales
  • Search: Filter by name or category
  • History: Saves your last 100 transactions
